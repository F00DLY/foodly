## Getting Started

First, run the development server:

run any command in the terminal in directory

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

then run in another terminal

```bash

cd server

npm run start

```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
